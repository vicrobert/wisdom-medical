医生简介

何盛华，女，本科，1977年毕业于白求恩医科大学医疗系。曾在江西医学院第一临床学院儿科、常州市三甲医院等地进修学习。从事儿科临床工作30余年，有丰富的临床经验发表省级论文2篇。市级科研成果二项。擅长儿科呼吸及消化等疾病的诊治。
